Le module blink permet de piloter la LED USB Blink(1) (issue d'un projet KickStarter) en envoyant des commandes HTTP au petit logiciel Blink1Control.

API

Voici la liste des paramètres du plugin. Dans tous les cas, un timout de 10s éteint le Blink.

colorsetColor()Set a static color
blinkblink()Blink a color to black
warnblink()Blink 2 colors (must use 2 parameters)
durationdefault: 1000Available for blink and warn

Rules

Before: 
if (options.confidence >= 0.9){
  SARAH.call('blink', {'color':'00FF00'});
} 
else if (options.confidence >= 0.8){
  SARAH.call('blink', {'color':'0000FF'});
} 
else {
  SARAH.call('blink', {'color':'FF0000'});
}
After: 
setTimeout(function(){ 
  SARAH.call('blink', {'color':'000000'});
}, 1000);
